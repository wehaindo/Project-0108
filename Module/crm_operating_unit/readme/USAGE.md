

To use this module, you need to:

1. Go to CRM > Sales > Leads or CRM > Sales > Opportunities
2. When creating a new lead or opportunity, the operating unit will be automatically set based on:
   - The operating unit of your default sales team, or
   - Your default operating unit if the sales team doesn't have an operating unit

3. You can filter leads and opportunities by operating unit using the search filter

4. You can only see leads and opportunities from operating units you have access to

5. The operating unit field is only visible if you belong to the "Multi Operating Unit" group
