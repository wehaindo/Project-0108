üì± Usage Scenarios

Scenario 1: Multi-Store Replenishment

Store ‚Üí DC ‚Üí Supplier Flow:

  • Store creates stock request tagged with Store OU
  • DC checks inventory with DC OU tag
  • If stock available: Create internal transfer
  • If stock not available: DC purchases from supplier
  • Full operating unit tracking throughout

Scenario 2: Department Stock Management

Restaurant Example:

  • Kitchen OU requests ingredients
  • Bar OU requests beverages
  • Retail OU requests merchandise
  • Central warehouse fulfills by department
  • Reports separated by operating unit

Scenario 3: Multi-Branch Operations

Franchise Model:

  • Each branch has own operating unit
  • Stock requests tagged by branch
  • Shared warehouse distribution center
  • Accounting and reporting per branch
